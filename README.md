Cli
---
configure cli script in json file of the module

`
"cli" : {
    "darwin" : "darwin.php"
}
`

run with:

 `php index.php -s <scriptname>`

Composer Install
----------------

Install composer

`curl -sS https://getcomposer.org/installer | php`

run composer.phar update
